---
title: Ownership Boundaries
---

Applies to: `Kairos and Aletheia`

Use Kairos documentation when the question is about:

- ingestion contracts
- identity resolution
- profile persistence
- audience logic
- API behavior

Use Aletheia documentation when the question is about:

- dashboard navigation
- analytical views
- inspection workflows
- operator tasks built on top of Kairos data

If a page mixes both, it should say so explicitly and describe the handoff between the platform and the dashboard.
