---
title: Understand the Platform-Dashboard Boundary
---

Teams usually get into trouble when they mix platform responsibilities and dashboard responsibilities.

## Healthy split

Use Kairos documentation when the question is about:

- data contracts
- identity resolution
- profile storage
- segment definitions
- API behavior

Use Aletheia documentation when the question is about:

- how people inspect data
- how operators move through workflows
- how dashboard views expose audience and profile state

## Why this matters

If the docs blur this boundary, users stop understanding where bugs, ownership, and operational responsibility actually live.
