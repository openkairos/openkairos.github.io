---
title: Aletheia Overview
---

Applies to: `Aletheia`

Aletheia is the dashboard and operator interface for Kairos.

It gives analysts, operators, and growth teams a visual working surface for:

- exploring profile data
- reviewing audience state
- checking analytics views
- following operational workflows tied to the underlying platform

Use Aletheia documentation when the question is about how people inspect, analyze, or operate on top of data that Kairos already manages.
