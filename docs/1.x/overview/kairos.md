---
title: Kairos Overview
---

Applies to: `Kairos`

Kairos is the system of record for customer data operations inside the product suite.

It owns:

- data ingestion
- identity resolution
- profile storage
- audience and segment evaluation
- activation-oriented APIs and downstream contracts

Use Kairos documentation when the question is about data movement, business rules, storage semantics, execution behavior, or integration surfaces.
