---
title: Profile Model
---

Applies to: `Kairos and Aletheia`

Profiles are the shared surface between the platform and the dashboard.

Kairos owns how profiles are created, updated, and stored. Aletheia helps users inspect and reason about those profiles.

Profile data may include:

- static attributes
- historical activity
- lifecycle state
- consent metadata
- segment memberships
- derived traits and scores
